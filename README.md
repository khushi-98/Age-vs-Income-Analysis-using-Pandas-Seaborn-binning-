# 📊 Age vs Income Analysis using Pandas & Seaborn

## 🚀 Project Overview
This project analyzes the relationship between **Age** and **Income** using Python. It demonstrates data grouping, visualization, and basic analysis.

---

## 🛠️ Technologies Used
- Python 🐍  
- Pandas 📊  
- Matplotlib 📈  
- Seaborn 🎨  

---

## 📂 Dataset
The dataset is manually created and includes:

- **Age**: 22 to 68  
- **Income**: 30,000 to 100,000  

---

## ⚙️ Steps Performed
1. Create DataFrame using Pandas  
2. Group Age into categories  
3. Group Income into levels  
4. Visualize data using plots  
5. Analyze trends using groupby  

---

## 📊 Visualizations

### 🔹 Age Group Distribution
- Countplot showing distribution of age groups  

### 🔹 Income Distribution by Age Group
- Boxplot showing income spread across age groups  

---

## 📈 Analysis
- Income generally **increases with age** in this dataset  
- Older age groups tend to have **higher average income**
